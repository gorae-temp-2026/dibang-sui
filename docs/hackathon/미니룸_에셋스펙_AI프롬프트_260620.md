# 모이가모인곳 미니룸 — 에셋 스펙 + AI 생성 프롬프트 팩 (2026-06-20)

> 목표: 싸이월드 미니미/샵/꾸미기 **퀄리티**의 인터랙티브 미니룸. SUI 해커톤 우승용 = product-grade.
> 전략: **코드(PixiJS) ↔ 아트(에셋) 분리.** Claude Code는 placeholder로 시스템 먼저 완성, 에셋은 병렬 준비 후 슬롯 교체.

---

## 0. 기본(default) vs 샵(꾸미기) 구분 — 중요
- **기본 모이 캐릭터 = 무료 기본 제공.** 사는 게 아니다. 꾸미지 않아도 그 사람의 **프로필 조회 가능**해야 한다.
- **방(기본 룸) = 기본 제공.**
- **샵 = 그 위에 얹는 꾸미기**: 아이템(가구·장식) + 모이 옷/액세서리. 요네로 구매.

---

## 1. 에셋 소싱 경로

### A. AI 생성 (1순위 — 스타일 일관성 = product-grade의 핵심)
방·모이·아이템·옷을 **하나의 파스텔 스타일로 통일**. 팩 혼용은 조잡해 보임. `캐릭터풍`(아스가르드) 레퍼런스로 스타일 잠금.
- Nano Banana (Google Gemini image) — 레퍼런스 이미지 스타일 고정에 강함.
- SEELE AI — 무료 + 상업 라이선스, 투명배경 PNG.
- BasedLabs / Ludo.ai / Easy-Peasy.AI — 레퍼런스 업로드 일관성.

### B. 유료 마켓 (저렴·즉시 — 꾸미기 구조 기성품)
- Unity Asset Store "160 Chibi Characters (customizable)" — 옷 교체형 치비. 모이 꾸미기에 적합.
- CraftPix Chibi Character Sprites — 저렴/구독.
- GraphicRiver(Envato) Chibi game assets — 단건 $5~20.
- Indoor 2D-Isometric Pack(899), GameDev Market 2D Interior — 방·가구.

### C. 무료 CC0 (라이선스 안전, 스타일은 인디룩)
- itch.io isometric/interior(free), OpenGameArt Furniture Kit, Kenney.nl(CC0).

### 라이선스 체크
- 팩별 **상업 사용 허용** 확인 필수. CC0 = 안전. AI는 상업 라이선스 명시 툴.

### 추천 조합
- **방·아이템·기본 모이 = AI 생성 통일** + (옷 교체 구조 필요 시) Unity 160 Chibi 보조.

## 4. 권장 워크플로 (D-2)
1. 기본 방 + 기본 모이 먼저 생성 → **스타일·캐릭터 레퍼런스 확정.**
2. 그 레퍼런스로 아이템 5종 + 옷 4종 생성(투명배경).
3. 병렬로 Claude Code는 PixiJS 미니룸 시스템을 **placeholder**(임시 박스/이미지)로 구축 — 샵 구매→배치, 요네 차감, 옷 장착, 모이 클릭→프로필.
4. 에셋 나오는 대로 placeholder 슬롯 교체 → product-grade 점프.
5. 質 > 量: 방1·기본모이1·아이템5·옷4면 데모로 충분. 폴리시에 집중.
