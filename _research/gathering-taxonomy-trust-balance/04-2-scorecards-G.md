1. **계모임의 일관된 특이성** — G1 폐쇄(초대)=신용 심사, G2 성취(이탈=신뢰 파산), G4 EM 원형, G5 갱신형, S3 순번 수령. 모든 축에서 "신뢰의 금융화" 시그니처가 반복됨. 분류 체계 전체에서 결혼식(이벤트)과 계모임(집단)이 대칭적 특이점.
2. **동질성 축의 양극** — 군대(공유물 없는 상황)와 가족(변경 불가능한 혈연) 사이에 변경 가능성 스펙트럼이 깔림: 혈연 < 신앙 < 학연(화석) < 지연 < 직연 < 취향 < 이해관계(즉시 해소).
3. **G4 복합 판정 6/14** — 절반 가까이가 단일 Fiske 모드로 운영되지 않음. 집단은 "지배 모드 + 보조 모드"의 합성이 정상 상태 — Fiske 모델이 관계(쌍) 단위 개념이라 집단 적용 시 혼합이 필연.
4. **이탈 비용의 독립성** — DAO(매도 즉시)와 계모임(신뢰 파산)은 둘 다 성취 멤버십인데 이탈 비용이 양극단. 귀속성 축과 별개로 "이탈 비용" 파생 지표가 신용 담보력의 대리 변수가 될 수 있음.
