# 06. 관계 분류 — R축 (1층 관계 빈칸 닫기)

> 1층에서 이벤트(E축)·집단(G축)은 분류했지만 **관계는 "결과물(엣지)"이라며 뺐다**(03). H8("관계 = 2인짜리 집단")·H10-b가 "관계도 G축의 2인 버전으로 분류 가능"이라 지목 → 그 **R축(R = Relation)**을 여기서 채운다. 이것으로 1층의 마지막 빈칸이 닫힌다.
>
> 핵심: 신뢰 네트워크 = 관계(엣지)의 그래프이므로, 엣지를 분류하는 R축은 다른 축보다 신뢰 모델에 더 직접적이다. 특히 R2 운영장부(CS/EM/MP/AR)는 Fiske가 **원래 쌍(관계) 단위로 정의**한 것이라, 집단(G4)에 쓸 땐 근사(△)였고 관계(R)엔 제자리다.

## 1. R축 (6개)

| 축 | 값 | 출처 | 비고 |
|---|---|---|---|
| **R1 형성** | 귀속 / 선택 | G2 귀속성 | 태어나며 주어짐(가족) vs 고름(친구·연인) |
| **R2 운영장부** ★ | CS / EM / MP / AR | G4 운영논리(Fiske, 쌍이 본가) | 관계 종류의 중심축. H12의 EM 잔액·CS 유대 구분이 여기 |
| **R3 대칭** | 대칭 / 비대칭 | (Parsons·위계) | R2와 독립(데이터 확인 §4). "AR ⇒ 비대칭" 단방향 상관 |
| **R4 지속** | 무기한 / 기한 / 갱신형 | G5 기한 | |
| **R5 독점** | 독점 / 비독점 | G1 경계(2인 변형) | **희소 축** — 부부·연인만 활성. 배타적 친밀 식별용 |
| **R6 연(緣)** | 혈연/지연/학연/직연/취향/신앙/이해/매력/상황 | G3 동질성 | 무엇으로 엮였나. 극단값은 타입 고정(혈연=가족·매력=연인), 중간은 자유 변동 |

R6 값 풀이: 혈연(피)·지연(지역)·학연(학교)·직연(직장/직업)·취향(관심사)·신앙(종교)·이해(이익)·매력(끌림)·상황(우연한 처지).

## 4. 발견

1. **R2 장부가 관계 타입의 중심.** CS(가족·부부·의형제) / EM(친구·동창·동료·선후배) / MP(거래·임대·전문가) / AR(사제·고용)으로 갈림. Fiske 4모델이 관계에선 근사(△) 없이 제자리.
2. **R3 ⊥ R2 (독립, 데이터 확인).** "비대칭인데 AR 아닌" 반례 = 임대인-임차인(비대칭의 출처=소유), 채권자-채무자(출처=채권). 둘 다 서열이 아닌 자원·금융 위치에서 비대칭이 옴 → R3은 R2에서 파생되지 않음. 단 한 방향 상관 **"AR ⇒ 비대칭"**(AR 3개 모두 비대칭)은 성립, 역은 불성립.
3. **R1 ⊥ R2 (독립).** 동창은 **귀속(졸업하면 자동)인데 EM**, 가족은 귀속인데 CS → 형성 방식이 장부를 결정하지 않음.
4. **R5 독점은 희소.** 21개 중 부부·연인 2개만 활성 → H10-b의 "G1 경계는 2인에서 소멸/변형" 확인. 버리지 않고 배타적 친밀 식별용으로 유지(계모임 순환수령을 빈도 1에도 남긴 것과 같은 논리).
5. **EM ↔ CS 전환이 타입으로 드러남.** 연인(EM→CS)·단골(MP→EM)처럼, H12의 "호혜 반복이 유대로 굳음"이 관계 종류 안에 시간 발전으로 박혀 있음.
6. **팬-스타가 "양자 상호" 가정을 깬다.** 다른 20개는 둘이 서로 아는 쌍방 관계인데, 팬-스타는 **한쪽만 아는 일방 관계**(팬은 스타를 알지만 스타는 팬을 모름). 신뢰적으론 일방 투자(조공). → 관계가 항상 대칭적 양자라는 전제의 예외.

## 5. 1층 닫힘 · 2층 연결

- 이로써 1층 분류 대상이 **이벤트(E)·집단(G)·관계(R)** 세 층위 모두 채워졌다. (장소는 좌표라 분류 제외 — 그대로.)
- 2층 연결: 관계는 fold의 **출력 상태**다(H8). R축은 그 출력이 어떤 종류인지를 분류하는 자 → **fold가 만들어낼 수 있는 관계 종류의 목록**이 R축 좌표 공간. 그리고 R2 장부 = 그 관계를 어떤 누적 연산으로 fold할지 지정(H10-a: EM→아벨군, AR→poset). 즉 R축이 2층 fold 연산의 분기표가 된다.

## 6. 열린 항목

- **R6 연(緣)의 지위** — 타입 축으로 유지하기로 결정(2026-06-15)하되, 중간 타입(친구·동료)에서 자유 변동하는 성격은 남음. "혈연⇒가족·이해⇒MP" 같은 연→다른축 상관은 추후 규칙화 후보.
- **R3 비대칭의 출처 분화** — 서열(AR)·소유·채권·전문성 등 비대칭 원인이 여럿. 신용 모델에서 이들을 구분할지 미정.
- **팬-스타류 일방 관계** — 양자 대칭 전제의 예외. 일방 엣지를 fold·전파(Φ)에서 어떻게 다룰지 2층 숙제.
