# SUMMARY — 모임 택소노미 (1p)

> 2026-06-10 | 신뢰 네트워크 정량화를 위한 "2인 이상 모임" 분류 체계. 브레인스토밍(축 15개 + 실제 모임 ~70개)을 4단계로 정제.

## 전체 지도 — 시스템 4층 사다리 (분류 → 누적 → 환산 → 상품)

이 리서치가 전체 시스템 어디에 있는지의 좌표. **"4층"이 두 뜻이라 주의** — 아래는 *시스템 단계*이고, "한 줄"의 "4층위(이벤트·집단·장소·관계)"는 *1층 안에서 모임을 분해*한 별개 개념이다.

```
1층  분류     모임 → 타입/프로필           classify          [집합론]        ✅ 완성   01~04(이벤트·집단) + 06(관계 R축) + 정적모델 v0.1
2층  동역학   액션 쌓아 관계·집단 만들기     상태 = fold(신호)  [집합론+대수]    ✅ 골격   07 fold 정의·검증 + 08 시뮬(10명10년) (H8 루프·실데이터 미착수)
3층  신용     관계망 → 신용 점수           Φ: 관계 → 신용     [범주론]        🔧 시제품 09 자원별 PageRank+행동신호+합산 → 노드 신용(sim). 정식 Φ는 𝒲 트랙
4층  금융     신용 → 실제 상품(대출·보증)   (서비스)                           ⬜ 미구축
```
- 번호의 엄밀성: 1·2층은 본 리서치가 직접 다룸, 3·4층은 hackathon 트랙 소관이라 느슨하게 붙인 번호. 핵심은 **분류→누적→환산→상품** 순서.
- 현재 위치: **3층 시제품**(09 — 그래프→PageRank→노드 신용 점수 작동, 영희·민수 상위·지훈 default 하락·현우 고립 하위). 결정값(무방향·페널티·비중)은 first-cut 임의. 정식 Φ·요네다·실데이터는 다음.

## 한 줄
"모임"은 단일 개념이 아니라 **이벤트/집단/장소/관계의 4층위**이며, 코어 17축 + 함의규칙 7개로 분류하면 7개 클러스터가 드러나고, **결혼식은 5개 축 값의 유일 조합(통과의례×전체소집×부조×공식기록×일생1회)으로 "네트워크 스냅샷" 전제가 형식화**된다.

## 핵심 결론 5개

1. **층위 분해가 최상위 분류다.** 원본 목록이 한 체계에 안 들어가던 이유 = 이벤트(결혼식)·집단(회사)·장소(버스)·관계(친구)가 섞여 있어서. 집단은 이벤트를 반복 생성, 이벤트의 누적이 관계(=신뢰 엣지)를 만든다. 장소는 좌표일 뿐.
2. **축에도 층위가 있다.** A(모임 속성)/B(참가자별 함수: 기쁨·가고싶음)/C(관찰시점: 과거·미래)를 분리. B는 버리는 게 아니라 "참가자×모임 사상의 속성"으로 보존 — 이것이 신뢰 네트워크의 원천 데이터.
3. **코어 17축 확정 (현 버전: v2)** (이벤트 7 + 집단 6 + 공용 4). 원본 축 4쌍 합침(유료→금전흐름, 혼성→동질성, 의도→Goffman 초점성, 가고싶음→강제성), 3축 쪼갬, 1축 폐기(과거/미래=timestamp). 이론 보강: Fiske 운영논리(CS/AR/EM/MP)·Collins 경계·Parsons 귀속성·기록성. 〔v2 2026-06-15: 값 목록 확장 — S1+귀속·S2 비대칭 정식화·S3+순환수령·G3+신앙/이해관계·G5+갱신형. 축 개수 17 불변. → [AXES-V2-CHANGELOG](./AXES-V2-CHANGELOG.md)〕
4. **7개 클러스터**: K1 의례-부조(스냅샷 생성기) / K2 호혜원장(계·품앗이=𝒲 본진) / K3 시장접촉(제도가 신뢰 대체) / K4 과업조직 / K5 unfocused 공존(**모임의 영점**) / K6 약유대 생성기 / K7 쌍 생성기(엣지 추가 연산, 주선자=신용보증 원형).
5. **범주론 접속점(스케치)**: Fiske 4모델 = 관계 Hom의 **enrichment 베이스 선택**(CS=진리값, AR=poset, **EM=아벨군** ← Pacioli=Grothendieck군과 접속, MP=ℝ). **EM만 미청산 잔액을 지원하므로 신용 함자 Φ의 정의역이 EM권인 이유가 설명됨.** 결혼식 = Hom(−,h)의 일생 1회 전수 샘플링 장치.

## 주요 누락 발견 (브레인스토밍에 없던 것)
**계모임**(신뢰=금융의 직접 사례, 최대 누락), **종교 예배/공동체**, 품앗이, 제사, 면접, 재판, 선거(U인데 집합 선택 — 규모·방향성 분리 증명), 자조모임(익명+최고친밀 — 익명성·유대 독립 증명), DAO, 라이브 방송 후원(역방향 금전).
